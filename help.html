Help Center

Find answers to common questions about DO FIT Browser.

How do I add a website to the homepage?

Open the Site Manager, click Add Site, fill in the required information and save.

How do I remove a website?

Open the Site Manager, select the website and click Delete.

How do I restore the default websites?

Open Settings and click Restore Defaults.

Where are my downloads saved?

Downloads are stored in the folder selected in Settings.

How does Allowed Domains work?

When you visit a new website, the browser asks whether to allow it once or permanently. Permanently allowed domains are stored locally.

How do I clear the browser cache?

Go to Settings and click Clear Browser Cache.

How do I report a bug?

Please create an issue on our GitHub repository with as much detail as possible.

Need more help?

Visit our GitHub repository or create an Issue.
